# 工作区主编排说明

## 执行前提（headless 与本轮 run）

- **`run_id` 唯一来源**：只认当次 **`.cursor_task/{run_id}/task_context.json`** 内 **`run_id` 字段**；所有路径、`outputs`、`run_completion.log` 与该目录绑定。**禁止**把工作区内其它 `.cursor_task/<其它 uuid>/` 当本轮任务。**禁止**用会话记忆或控制台输出里的别组 UUID 替代。
- **`headless`**：通过 **`agent`** 在非交互上下文执行时不存在「编辑器已注入 rules」的保证；编排以 **磁盘上** 本文件与 **`rules/`** 为准。**不得**弱化顺序：先 **`task_context.json`** → 再其中 **`qa_rule_file`** → 再按需对照本文件与其余 **`rules/`**。
- **逐步操作与路径级细则**：以当轮磁盘 **`.cursor_task/{run_id}/task_prompt.md` 内 Task 1–6** 为**唯一细纲**；本节只给原则，**不**与任务书逐步重复。

## 主职责（原则）

- **主编排链**（与任务书 Task 2–6 同序，不另起一套）：拉取正文 → QA 抽取与 CSV → 质检 → **`dify_upload`** → **`run_completion.log`**。
- **`dify_upload`**：使用 `from dify_upload.resolve_target import resolve_dify_target`（或与 `from dify_upload import resolve_dify_target` 等价的包导出），用本轮 **`task_context`** 与工作区 **`.env`** 得到 **`DifyTargetConfig`**。**禁止**手拼 **`DIFY_TARGET_*`**、凭空填 **`api_base`/`api_key`**。
- **上传/占位/是否对真实知识库执行写入**：以当轮 **`task_prompt` Task 5** 与 **`task_context.json`**、QA 规则为准；**不**在本节展开布尔分支。
- 未达标时按规则决定是否重试、修复、降级或停止。

## `feishu_fetch` 与落盘（原则）

- **`ingest_kind`**、**`resource_plane`**、**落盘根目录**须与同目录 **`task_context.json`** 及当轮 **`task_prompt.md`（Task 2 与文末 `feishu_fetch` 段）**一致；缺字段或与 JSON 矛盾则停止说明。
- **`output_dir`** 不向 JSON 索要字段名占位；解析后须为工作区根下 **`.cursor_task/{run_id}/outputs`**——与任务书 Task 2 一致。

## 主流程原则

1. 不伪造工具结果，不预设外部已为真。
2. 先取正文/结构再进 QA。
3. 仅在需要 QA 时套用命中规则。
4. QA 只约束抽取与 CSV 质检，不替代整条链路。
5. 脚本生成 QA 时仍须同时满足 **`rules/qa/base/*`** 与 **`task_context.json` 所指 `qa_rule_file`** 整包；不得只命中「数量/条数」单点。
6. 未达标时按规则决定重试、修复、降级或停止。

## 长期资产与任务产物

- **只读（勿写入本题产出或擅自改文件）**：`AGENTS.md`、`rules/`、`tools/`（含 **`dify_upload/`**、**`feishu_fetch/`**）、`runtime/webhook/`、`vla_env_contract/`。
- **本轮磁盘产物**：**`task_prompt.md`**（落盘时已预写为本轮入口说明，**除非任务书写明可改否则视为只读**）、**`task_context.json`**、**`outputs/`**、收尾 **`run_completion.log`**。**禁止**把临时/结果写回 **`rules/`** 等长期目录。

## 失败与停止

须停止并说明原因，例如：

- 文档不可用或 **`task_context` 不足以继续**
- QA 不达标且不许降级
- **上传用 CSV**不符合当轮 **`task_prompt` Task 3–4** 与同轮 **`rules/qa/base/output_contract.mdc` / `quality_checks.mdc`**
- 上传失败且重试前提不成立

- **禁止**空话、**禁止**伪造已成功。
